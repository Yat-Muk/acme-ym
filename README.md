## VPS本地IP的acme證書申請一鍵腳本

--------------------------------------------------------------

## 一鍵腳本
```bash
bash <(curl -Ls https://raw.githubusercontent.com/Yat-Muk/acme-ym/master/acme_pro.sh)
```
--------------------------------------------------------------

## 快捷命令： ac-pro

--------------------------------------------------------------

## 功能與特點：

1：支持純IPV4、純IPV6、雙棧VPS

2：支持80端口模式與DNS API模式，支持單域名與泛域名

3：支持Cloudflare/騰訊DNSPod/阿里Aliyun托管解析平台的DNS API申請

4：查詢、撤銷並刪除當前已申請的域名證書

5：手動一鍵或者指定續期域名證書

6：證書文件默認存放在root/cert文件中

--------------------------------------------------------------

注意：使用80端口模式時，會強制釋放80端口，且不建議與nginx、caddy類帶有自動申請證書的應用同時使用
